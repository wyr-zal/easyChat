# Excel 个性化群发

## 用途

新增一个独立窗口和独立入口脚本，用于从 Excel 中读取联系人数据，按 `{{列名}}` 占位符渲染个性化消息，并通过 EasyChat 现有的微信自动化能力逐个发送。

## 启动方式

```powershell
python excel_sender_gui.py
```

或直接双击：

```text
start_excel_sender.bat
```

## 打包方式

新增了独立打包脚本：

```powershell
python pack_excel_sender.py
```

默认会调用 `PyInstaller`，打包成单文件窗口程序，输出名称为 `EasyChatExcelSender`。

常见用法：

```powershell
# 默认：单文件、无控制台窗口
python pack_excel_sender.py

# 输出为目录模式
python pack_excel_sender.py --onedir

# 指定程序名称
python pack_excel_sender.py --name MyExcelSender

# 打包时保留控制台
python pack_excel_sender.py --console
```

打包完成后，产物会出现在：

- 单文件模式：`dist/EasyChatExcelSender.exe`
- 目录模式：`dist/EasyChatExcelSender/`

## 数据要求

- 默认支持 `.xlsx`
- 同时兼容 `.xls` 和 `.csv` 的读取入口，但优先建议使用 `.xlsx`
- Excel 必须存在你在界面中配置的发送识别列，默认是 `微信号`
- 发送识别列也可以改成 `姓名`、`备注`、`显示名称` 等
- 其余列可自由作为占位符字段使用

示例列：

- `微信号`
- `姓名`
- `课程名`
- `到期时间`

## 模板规则

模板语法固定为：`{{列名}}`

示例：

```text
您好 {{姓名}}，您的课程 {{课程名}} 将于 {{到期时间}} 到期，请及时处理。
```

发送时，系统会按 Excel 每一行的数据替换对应占位符。

## 界面结构

当前窗口已调整为 3 个分页标签：

1. `基础设置`
2. `数据与模板`
3. `发送执行`

其中：

- `数据与模板` 页支持 Excel 读取、正则筛选配置、模板编辑
- `发送执行` 页支持发送控制、发送预览、发送日志
- 发送预览表中的前 10 条记录支持手工编辑和删除

## 界面流程

1. 在 `基础设置` 中确认微信快捷键和语言
2. 切到 `数据与模板`，选择或拖入 Excel 文件
3. 在同页确认“发送识别列”，默认是 `微信号`，也可以改成 `姓名`、`备注` 等
4. 点击“读取数据”，确认识别到的列名中包含当前发送识别列
5. 如需筛选联系人，在同一页配置筛选字段、正则规则并点击“应用筛选”
6. 在同一页输入消息内容并检查占位符状态
7. 切到 `发送执行`，设置发送间隔
8. 点击“刷新预览”查看前 10 条渲染结果
9. 如需细调，直接在预览表中编辑单行内容，或点击每行后的“删除”
10. 点击“开始发送”执行逐个发送，并查看发送日志
11. 如需中断，在同一页点击“停止发送”

## 发送识别列说明

- 发送时，系统会把“发送识别列”这一列的值粘贴到微信搜索框中
- 默认使用 `微信号`
- 如果你的业务表更适合按姓名、备注或显示名称发送，可以直接改成对应列名
- 如果当前 Excel 里不存在这个列，读取和发送前校验都会阻止继续

## 正则筛选说明

- 默认筛选字段为 `场景`
- 支持自定义多个字段，字段之间用英文逗号分隔，例如：`场景,标签,备注`
- 支持标准 Python 正则表达式
- 可勾选“忽略大小写”
- 点击“重置筛选”可恢复当前已加载的全部发送数据

常见规则示例：

- 到期提醒：`到期|续费|复购`
- 跟进客户：`待跟进|未回复|沉默`
- 地区筛选：`上海|北京|深圳`
- 精确匹配：`^高意向客户$`
- 排除空白：`^(?!\s*$).+`

## 注意事项

- 新模块不会修改原 EasyChat 主窗口逻辑
- 发送依旧依赖 PC 微信桌面客户端和 `Ctrl+Alt+W` 显示/隐藏窗口快捷键
- 如果模板中用了不存在的列，占位符校验会阻止发送
- 如果某一行当前发送识别列的值为空，该行会被自动跳过
- 如果在预览表里手工修改了单行消息，该行发送时会优先使用手工内容
- 如果重新读取 Excel，当前筛选结果和预览表手工调整会按新数据重新生成
- 当前发送结果以执行过程日志为准，不依赖旧版聊天记录校验返回值
- 模板编辑区为了兼容拼音输入法，改为延迟保存；要看最新渲染结果请点击“刷新预览”
- 微信启动说明不再在程序启动时自动弹出，如需查看可在 `基础设置` 页手动打开
